How to run the Vehicle Service Center Management System (VSCMS) Project
1. Access the System: Open your web browser and navigate to the server address (e.g., http://localhost:8000).
2. Login: Use the provided credentials to log in as an admin or create a new account.
3. Manage Users: Admin can add, edit, or remove staff and customer accounts.
4. Book Appointments: Customers can book appointments through the web interface.
5. Service Management: Staff can view and update service records, manage inventory, and generate invoices.
6. Reports and Feedback: Admin can generate reports and view customer feedback. 
7. Open PHPMyAdmin (http://localhost/phpmyadmin)
8. Create a database with name vscmsdb
9. Import vsmsdb.sql file(given inside the zip package in SQL file folder)
10. Run the script http://localhost/vscms (frontend)
11. For admin panel http://localhost/vscms/admin  (admin panel)
